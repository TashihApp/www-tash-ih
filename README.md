# tashih.app

Landing page and privacy policy for [Tashih](https://tashih.app), an AI-powered Turkish writing enhancement app for macOS.

## Structure

- `index.html` - Landing page with product overview, pipeline, use cases, features, and pricing
- `privacy.html` - Privacy policy
- `CNAME` - Custom domain configuration for GitHub Pages

## Deployment

Static site hosted on GitHub Pages. Push to `main` to deploy.

## Development

Pure static HTML with embedded CSS and vanilla JS. No build step required. Open `index.html` in a browser to preview.
